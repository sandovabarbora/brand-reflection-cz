# Brand Reflection Series

A series of four artifacts that measure Czech advertising from three
sides: how it looks, how it sounds, where it stands. Plus one tool
that combines them into a single score for any video URL.

**Live hub:** https://sandovabarbora.github.io/brand-reflection-cz/

| | Live | Repo |
|---|---|---|
| Episode 1 — Color fingerprint | [link](https://sandovabarbora.github.io/color-fingerprint-cz/) | [repo](https://github.com/sandovabarbora/color-fingerprint-cz) |
| Episode 1.5 — Sound fingerprint | [link](https://sandovabarbora.github.io/sound-fingerprint-cz/) | [repo](https://github.com/sandovabarbora/sound-fingerprint-cz) |
| Episode 2 — Strategic topography | [link](https://sandovabarbora.github.io/strategic-topography-cz/) | [repo](https://github.com/sandovabarbora/strategic-topography-cz) |
| Tool — Spot Scorer | [link](https://sandovabarbora.github.io/spot-scorer/) | [repo](https://github.com/sandovabarbora/spot-scorer) |

This repo is the hub: one static HTML page that introduces the series,
links to each episode, and walks through one worked example
(Pilsner Urquell's *Alfons* spot) measured across all four lenses.
